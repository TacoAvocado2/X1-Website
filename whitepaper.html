Version 1.0

1. Abstract

X1 Coin is a decentralized digital currency powered by the SHA256 hashing algorithm and secured through a Proof-of-Work (PoW) consensus mechanism. It provides a fast, secure, and transparent financial network with a fixed supply and predictable issuance schedule.

2. Introduction

X1 Coin removes reliance on centralized financial authorities by enabling a peer-to-peer network secured by cryptographic proof instead of trust.

Official Website: https://x1coin.net

3. Technical Overview

Consensus Mechanism: Proof-of-Work (PoW)

Algorithm: SHA256

3.1 Block Structure

Each block contains transactions, timestamp, previous block hash, and nonce.

4. Token Economics

Total Supply: 21,000,000 X1

Pre-mined: None

Algorithm: SHA256

5. Emission Schedule

Initial Reward: 50 X1

Block Halving: Every 210,000 blocks

Rewards decrease by 50% each halving cycle.

6. Network Design

X1 operates on a decentralized peer-to-peer network where nodes validate transactions and miners secure the blockchain.

7. Mining System

Miners solve SHA256 puzzles to validate blocks and earn rewards plus transaction fees.

8. Security Model

Security is based on SHA256 hashing, distributed mining, and difficulty adjustment mechanisms that protect against attacks.
